Коли починаєш новий лесон переходиш на мастер (git checkout master)
(Має синеньким підствічуватись, якось так - (master) )
Дальше створюєш нову бренчу від пустого мастера 
(git checkout -b "Назва бренчі без лапок")
потім робиш завдання
дальше все як по старому - git add . -> git commit........АЛЕЕЕЕЕЕЕЕЕЕЕ
пушити  треба по іншому от так - 
git push origin "Назва тої бренчі яку ти створила перед цим"
